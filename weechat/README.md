# weechat

## Start, Exit

```bash
weechat
```

```weechat
/exit
```

## Server

```weechat
/server add ngircd localhost/6667
```

```weechat
/server list
```

## Nick

```weechat
/set irc.server.ngircd.nicks w67
```

```weechat
/nick w67
```

## Save

```weechat
/save
```

```weechat
/connect ngircd
```

```weechat
/join #bot
```
