# weechat

## Start, Exit

```bash
weechat
```

```weechat
/exit
```

```weechat
/server add ngircd localhost/6667
```

```weechat
/server list
```

```weechat
/set irc.server.ngircd.nicks w67
```

```weechat
/nick w67
```

```weechat
/connect ngircd
```

```weechat
/join #bot
```

```weechat
/exit
```
